How the AI Behaves

Replies instantly, 24/7 — no manual response needed
Always asks before assuming severity ("smoke" = asks if cooking or fire)
Detects tenant language — replies in English or French automatically
Sends a YouTube repair video for common fixable issues
Creates a ticket on every message — even the first one
Handles up to 3 exchanges per issue, then wraps up
